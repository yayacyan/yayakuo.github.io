# きくよう みんなの応援券 対象店検索

菊陽町が発行する「きくよう みんなの応援券」の対象店舗を分類・地区別に検索できるウェブサイトです。

## 主な機能

- キーワード検索（店名・住所・サービス内容・電話番号）
- 分類フィルタ（13種）＆クイックピルボタン
- 地区フィルタ（8地区）＆クイックピルボタン
- 並び順変更（登録番号・店名・地区・分類）
- 検索パネルの表示/非表示トグル（内縮）
- キーワードハイライト表示
- 分頁（12件/ページ）
- 先頭に戻るボタン
- レスポンシブデザイン（スマートフォン対応）

## ファイル構成

```
index.html        メインページ
css/style.css     スタイルシート
js/data.js        店舗データ（200件超）
js/app.js         検索・レンダリングロジック
```
